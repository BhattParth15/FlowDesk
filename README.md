A full-stack Task Manager web application built using React.js,Node.js,Express.Js and MongoDb with JWT authentication, role-based access control, real-time updates using Socket.io, and secure file upload support.

# Features

User Registration & Login (JWT Authentication)

Role-Based Access Control (Admin/User)

Create, Update, Delete Tasks

Staff Create, Any Task assign to Staff and Update Staff data so email send to staff email address.(use Nodemailer).

Task Status Management

Real-Time Updates using Socket.io

Secure File Upload (Images/Videos)

Pagination & Search Support

Permission-based UI Rendering
